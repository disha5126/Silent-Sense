# SilentSense

Wearable ESP32-based sound classification system using Edge Impulse.

## Features
- Real-time sound classification (danger, speech, music)
- RGB LED feedback
- Vibration motor alerts
- Blynk IoT notifications
- Adaptive thresholds

## Hardware
- ESP32 DevKit V1
- INMP441 I2S Microphone
- RGB LED
- Coin vibration motor
- LiPo battery (planned)

## Project Structure
- firmware → ESP32 code + model
- hardware → wiring & PCB (future)
- docs → notes & development logs
